# Diksha_Education_Analytics_Dashboard
This project presents a comprehensive analysis of digital education usage across India using Power BI. It focuses on user engagement, learning activity, and course consumption trends across states and over time.
The dashboard enables stakeholders to monitor platform performance, identify high-engagement regions, and track the effectiveness of digital learning initiatives.
Problem :
stakeholders need a centralized system to - 
•	Track user engagement across states 
•	Monitor learning activity trends 
•	Analyze course enrolment and completion rates 
•	Evaluate digital education adoption 
This dashboard solves these challenges by providing interactive and real-time insights.

Key Metrics :
User Metrics - 
•	Registered Users: 16.16M 
•	Daily Active Users: 58K 
Learning Metrics - 
•	Learning Sessions: 5,586M 
•	Learning Minutes: 64,560M 
Content Metrics
•	Energized Textbooks: 7582
•	E-Contents: 367K
Course Metrics -
•	Enrolments: 186M 
•	Completions: 147M 
•	Certificates Issued: 127M 
•	Total Courses: 613 

Dashboard/Report Features :
1. User Analytics
•	State-wise active users distribution 
•	Registered users growth trend (2018–2026) 
•	Last 3 months active user trend 
2. Learning Analysis
•	Learning sessions & minutes trend over years 
•	Engagement growth insights 
•	Content consumption behavior 
3. Content Insights
•	Total energized textbooks (7,871+) 
•	Medium-wise distribution (Hindi, English, Marathi, etc.) 
•	Grade-wise content availability 
4. Course Performance
•	Year-wise enrolments, completions, and certifications 
•	State-wise certificate distribution 
•	Course effectiveness tracking
